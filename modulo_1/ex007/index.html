Aqui vou fazer algumas listas com tags HTML5

Primeira é a Lista Ordenada ou Orderned List, para ela uso o tag <ol> e <li> para cada identificar cada item da lista e posso muda o type da Lista Ordenada.
Exemplo:

  1. Item 1
  2. Item 2
  3. Item 3

Segunda lista é a Lista não Ordenada ou Unordered Lists, para ela uso a tag <ul> e <li> como visto antes e podendo mudar o type dela também.

  • Item 1
  • Item 2
  • Item 3

Terceira lista é a Lista de Definição, é usada como dicionário, é usada a tag <dl> e cada termo é um <dt> e cada descrição é um <dd>
